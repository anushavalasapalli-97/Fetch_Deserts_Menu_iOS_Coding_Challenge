# Fetch_Deserts_Menu_iOS_Coding_Challenge

This Application displays the deserts on the Main Page. When you click on each desert, it will show the desert name, image, and ingredients, and measurement.

Architecture: MVVM
Language: Swift
UI_Design: Swift-UI
